# PMPP - Programming Massively Parallel Processors

This repository contains solutions and explorations of the various exercises from **Programming Massively Parallel Processors** (Fourth Edition).

## Purpose

This project is a learning journey to understand techniques for optimizing code to leverage the full potential of modern and future hardware. Through implementing solutions to PMPP exercises, I aim to develop deeper expertise in:

- Parallel programming concepts and patterns
- GPU computing and CUDA development
- Hardware utilization and optimization
- Performance profiling and analysis
- Scalable algorithm design

## Repository Structure

Solutions are organized by chapter and exercise. Each chapter includes:

- **Exercise implementations**: Code solutions with implementation details
- **Theory questions markdown**: A dedicated markdown file with solutions and answers to theory-based questions from the chapter
- **Notes and analysis**: Implementation notes and performance analysis where applicable
